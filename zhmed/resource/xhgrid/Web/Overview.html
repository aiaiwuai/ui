Key Features of XiaoHui Grid
  • Provides core functionalities, such as sorting, moving, merging cells, editing, filtering, searching, and Excel import / export.
  • Controls property for columns, rows and cells.
  • Allows users to edit data using web browser.
  • Row / Column merge and the auto-merge function are supported.
  • Automatic row height adjustment function.
  • Hierarchical data representation (show / hide).
  • Excel: Import (csv, xls, xlsx), Export (xls, xlsx), Copy and Paste between KoolGrid and Excel.
  • Automatic sum features: total, subtotal, change label, etc.
  • Selection control: single line, multiple lines, single cell, multiple cells.
  • Various renderers: CheckBox, Icon, Image, Spark Chart, etc.
  • Allows users to use HTML in headers and cells.
  • Provide columns to indicate if rows are added, modified, or deleted.
  • Footer.
  • Lazy Loading.
  • OLAP pivot function.
  • Web accessibility compliance.
Grid Layout and Data Integration
  • Creating a grid requires Layout and Data.
  • Layout is an XML-formatted file or string to define the grid's columns, object ID, properties, styles, etc.
  • Data supports JSON, Array, XML, CSV(comma-separated values) and TSV(tab-separated values) formats, and can be passed as a file (URL) or as a string.
